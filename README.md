## Installation

```bash
npm install -g code-hallucinator
```

## Usage

Run it on your project (make sure to commit first, this is destructive!):

```bash
hallucinate "**/*.{js,ts}"
```

Or output to a separate folder (preserves your originals):

```bash
hallucinate "**/*.{js,ts}" --out-dir hallucinated
```

Or try a dry run:

```bash
hallucinate "**/*.{js,ts}" --dry-run
```
